# mods-copper_tools
made by Intellij IDEA

### 情報
銅製の防具・道具を追加する。  
**内容**
* 銅のツール(剣、シャベル、ツルハシ、斧、クワ)
* 銅の防具(ヘルメット、チェストプレート、レギンス、ブーツ)
* 銅塊
* 各種レシピ(鍛冶台による防具の装飾は未対応)
